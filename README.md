# Qi Li - Personal Website

This is the source code for my personal academic website hosted on GitHub Pages.

## 🚀 Features

- **Responsive Design**: Works perfectly on desktop, tablet, and mobile devices
- **Modern UI**: Clean, professional design with smooth animations
- **Interactive Elements**: 
  - Publication tabs with filtering
  - Smooth scrolling navigation
  - Scroll spy for active section highlighting
  - Fade-in animations on scroll
- **Comprehensive Sections**:
  - About Me
  - Research Interests
  - Publications (Under Review, Journal, Conference)
  - Research Projects
  - Experience & Education Timeline
  - Awards & Honors
  - Technical Skills
  - Teaching Experience
  - Contact & Job Opportunities

## 📁 Project Structure

```
github-page/
├── index.html          # Main HTML file
├── styles.css          # CSS styling
├── script.js           # JavaScript for interactivity
└── README.md           # This file
```

## 🎨 Technologies Used

- **HTML5**: Semantic markup
- **CSS3**: Modern styling with CSS Grid, Flexbox, animations
- **JavaScript**: DOM manipulation, scroll effects, tab switching
- **Font Awesome**: Icons
- **Google Fonts**: Inter and Roboto Mono fonts

## 🚀 Deployment

This website is deployed using GitHub Pages:

1. Push this repository to your GitHub account
2. Go to Settings > Pages
3. Select the main branch as the source
4. Your site will be available at: `https://<username>.github.io/<repository-name>/`

## 📝 Customization

To customize this website for your own use:

1. **Update Personal Information**:
   - Edit the header section in `index.html`
   - Update contact information
   - Add your own social media links

2. **Update Content**:
   - Replace publications with your own
   - Update research projects
   - Modify skills and experience sections

3. **Customize Colors**:
   - Edit CSS variables in `styles.css` (lines 6-19)
   - Change primary colors, backgrounds, etc.

4. **Add/Remove Sections**:
   - Add new sections by copying existing section structure
   - Update navigation links accordingly

## 🎯 Key Features Explained

### Publication Tabs
The publications section uses JavaScript tabs to organize papers by status:
- Under Review
- Journal Papers
- Conference Papers

### Timeline Design
The experience section uses a vertical timeline with alternating left/right layout for better visual organization.

### Skills Grid
Technical skills are displayed in a responsive grid with hover effects and color-coded icons.

### Responsive Design
The website adapts to different screen sizes:
- Desktop: Full layout with sidebar timeline
- Tablet: Adjusted grid layouts
- Mobile: Single column layout with optimized spacing
